# Sales Performance Dashboard | Power BI

## Project Overview

This project focuses on building an interactive Power BI dashboard to analyze sales performance and generate actionable business insights.

The dashboard was developed using a sales transaction dataset and includes KPI tracking, customer analysis, regional performance analysis, and sales channel evaluation.

## Key KPIs

- Total Revenue
- Total Profit
- Total Transactions
- Average Order Value (AOV)
- Profit Margin (%)

## Analysis Performed

### Executive Overview
- Revenue Analysis
- Profit Analysis
- Transaction Monitoring
- KPI Tracking

### Customer Insights
- Top Customers Analysis
- Payment Method Distribution
- Customer Spending Patterns

### Business Performance Analysis
- Revenue by Category
- Revenue by Region
- Sales Channel Performance
- Profitability Analysis

## Dashboard Features

- Interactive Slicers
- Dynamic KPI Cards
- Bar Charts
- Donut Charts
- Customer Analysis Tables
- Business Intelligence Reporting

## Tools & Technologies

- Power BI
- SQL
- MySQL
- Data Analytics
- Business Intelligence

## Outcome

Developed an interactive dashboard that converts raw sales data into meaningful business insights to support data-driven decision-making.
